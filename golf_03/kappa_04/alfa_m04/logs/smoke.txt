usage: factory_order_emitter.py [-h] [--order-id ORDER_ID]
                                [--issued-by ISSUED_BY] [--target TARGET]
                                [--priority PRIORITY] [--timestamp TIMESTAMP]
                                [--summary SUMMARY] [--narrator NARRATOR]
                                [--extra-field EXTRA_FIELD] [--dry-run]
                                translator_payload destination

Emit validated factory-order payloads

positional arguments:
  translator_payload    File containing translator spike output
  destination           Where to write the factory-order JSON

options:
  -h, --help            show this help message and exit
  --order-id ORDER_ID   Override order identifier
  --issued-by ISSUED_BY
                        Override issuing workspace
  --target TARGET       Override target workspace
  --priority PRIORITY   Set priority flag
  --timestamp TIMESTAMP
                        Override timestamp (ISO-8601)
  --summary SUMMARY     Override lore summary
  --narrator NARRATOR   Set metadata.narrator_profile
  --extra-field EXTRA_FIELD
                        Inject arbitrary KEY=VALUE assignments (VALUE parsed
                        as JSON when possible)
  --dry-run             Preview payload without writing
